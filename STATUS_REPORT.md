## Project Overview

A hyperlocal community marketplace for Ennerdale, Johannesburg South. Sellers list products via a web form; buyers browse the grid and enquire directly via WhatsApp. The platform uses deep-link sharing so any listing can be sent as a WhatsApp broadcast that opens directly to the product card.

**Core Differentiators vs. Facebook Marketplace / Gumtree:**
- WhatsApp-native sharing (pre-composed messages, not just a link)
- Deep-link anchoring: a shared link scrolls + highlights the exact product
- AI-powered community assistant (Claude claude-sonnet-4-6 via Supabase Edge Function)
- Hyperlocal: Ennerdale only — no noise from other cities
- Free for community members

## Architecture Summary

```
Browser (GitHub Pages)
    │
    ├── Reads listings ──────────────────► Supabase DB (anon, RLS: SELECT only)
    │
    ├── Auth (login.html) ───────────────► Supabase Auth (email/password)
    │
    ├── Uploads images ──────────────────► Supabase Storage (bucket: product-images)
    │         └── Inserts listing row ──► Supabase DB (authenticated, RLS enforced)
    │
    ├── AI chat messages ────────────────► Supabase Edge Function (ai-chat)
    │                                           └── Anthropic API (Claude claude-sonnet-4-6)
    │
    └── WhatsApp deep links ─────────────► wa.me (direct, no server needed)
```
